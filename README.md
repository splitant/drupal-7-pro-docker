# Drupal 7 pro docker

## About The Project

The goal is to set up fastly a local Drupal 7 project with docker environment for professional uses.

### Built With

* [Docker4Drupal](https://github.com/wodby/docker4drupal)

## Getting Started

### Installation

   ```sh
   make create-setup <project> <repo-git>
   # Fill env file
   make up
   make drupal-install
   ```
